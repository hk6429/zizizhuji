# 字字珠璣 — 內容錨定檔（groundwork，不隨網站出貨）

這裡是給「國中／選手」兩級題庫**批次生成**用的**事實錨定資料**。
用途是把自寫題目對齊政府開放資料與權威詞頭，**避免捏造**。
瀏覽器端不會載入這些檔案，它們只服務出題流程的離線比對。

## 誠信鐵律

- 只用**事實層**：字音、筆畫、成語**詞頭**（皆為事實，可自由引用）。
- **絕不**抄任何辭典的釋義／例句／解析散文（受著作權保護）。
- 釋義、題幹、選項一律**自寫**，並標注來源「教育部＋g0v」。
- 湊不到目標數量（國中 4000 字／2000 成語、選手 7000 字／4000 成語）時，
  **老實回報實際題數**，不得為達標捏造。

## 檔案

### `ziyin-anchor.json` — 字音字形錨定（20,910 字）

形狀：`{ "字": { "strokes": N, "zhuyin": ["注音1", ...] } }`

範例：`"拓": { "strokes": 8, "zhuyin": ["ㄊㄚˋ","ㄊㄨㄛ","ㄊㄨㄛˋ","ㄓˊ"] }`

- 來源：**全字庫 CNS11643**（政府開放資料，中央研究院／國發會）
  - `CNS_phonetic.txt`（CNS 碼 → 注音）
  - `CNS_stroke.txt`（CNS 碼 → 筆畫）
  - `CNS2UNICODE_Unicode BMP.txt`（CNS 碼 → Unicode）
- 建法：以 Unicode 對接三表，篩 BMP 漢字（0x4E00–0x9FFF）。
- 用途：出「字音字形」題時，用來核對某字的**正確讀音集合**與筆畫，
  誘答選項可從**相近但錯誤**的讀音挑，確保「正解唯一、誘答似是而非」。

### `chengyu-anchor.json` — 成語詞頭錨定（3,008 條）

形狀：`{ "source": "...", "count": 3008, "idioms": ["一了百了", ...] }`

- 來源：**教育部重編國語辭典**（CC BY-ND）經 **g0v moedict** `dict-cat.json` 的「成語」分類。
- **只取詞頭清單**（事實），釋義與題目全部自寫。
- 授權：moedict 程式碼 CC0、資料 CC BY-ND；本專案僅引用**詞頭字串**（非受保護散文），
  並於題庫來源欄標注「教育部＋g0v moedict」。
- 缺口：3,008 條，距「選手級 4000 成語」尚差約千條；後續可補教育部《成語典》詞頭，
  仍以**詞頭＋自寫**原則辦理，不足就照實回報。

## 重建方式

原始全字庫／moedict 檔為公開下載，非本 repo 追蹤。需要重建時：
1. 全字庫：至國發會「政府資料開放平臺」下載 CNS11643 資料集，取上列三檔。
2. moedict：`g0v/moedict-data` 的 `dict-cat.json`，取 `成語` 分類 name 陣列。
3. 依上述 join／篩選規則產出本兩檔。
